AI Website Copy Generation System

AI + Prompt Engineering | Conversion-Focused Web Copy | Local Business Automation

---

Project Overview

This project demonstrates a structured prompt engineering framework designed to generate high-converting website copy for local service businesses.

Instead of producing generic AI text, this system uses clearly defined inputs, structured output formatting, and writing constraints to generate persuasive, website-ready content.

---

Business Case Study

Red Door Cafe – Coimbatore

Business Type: Cozy Cafe  
Target Audience: Students, freelancers, young professionals  
Brand Personality: Warm, relaxed, welcoming  

The system was used to generate complete website copy tailored specifically to this business.

---

Prompt Design Logic

The framework follows a modular structure:

1. Business Input Block
   - Business name
   - Location
   - Target audience
   - Brand personality
   - Services
   - Unique selling points
   - Conversion goal

2. Structured Output Requirements
   - Homepage copy (headline, benefits, trust section, CTA)
   - Services page content
   - Conversion-focused CTA sections

3. Writing Constraints
   - No generic phrases
   - No fluff or corporate jargon
   - Clear value propositions
   - Short, persuasive sentences
   - Tone adaptation based on business type

4. Refinement Layer
   - Strengthening value proposition
   - Improving clarity
   - Making CTAs more action-oriented

All prompts are available inside the `/prompts` folder.
